Create navigational links within a web site

There are several techniques to navigate within a web site. The sematic tag for navigation is the < nav> tag. You can navigate through your web site by using relative addressing. For our examples we will be using the unordered list tag(< ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the correct page
  • going to a specific place on the correct page (on page navigation)

Navigational menu are placed in your < header> semantic area. They may also be placed in your < footer> area as a sitemap. The nav tag like other semantic tag should have a < hx> tag. HOWEVER this header is optional. If you omit the header tag you will recive a document outliner error: Untitled Section. Until you are shown how to avoid this error message it will be acceptable on your document outline.
