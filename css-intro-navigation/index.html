Create navigational links within a web site

There are several techniques to navigate within a web site. The sematic tag for navigation is the < nav> tag. You can navigate through your web site by using relative addressing. For our examples we will be using the unordered list tag(< ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the correct page
  • going to a specific place on the correct page (on page navigation)

Navigational menu are placed in your < header> semantic area. They may also be placed in your < footer> area as a sitemap. The nav tag like other semantic tag should have a < hx> tag. HOWEVER this header is optional. If you omit the header tag you will recive a document outliner error: Untitled Section. Until you are shown how to avoid this error message it will be acceptable on your document outline.

CSS Coding Styles Summary

Browser Defaults (no css) you have no conrol over the styling
Inline this is for a single specific element (tag) instance. your css will be specified in a particular within the element. This css affects ONLY the element tag that contains the specification. style="rule; [rule; ...]" <style="color:red; width:50px"> ...</style>
Embedded the specification of css as embedded happens in your <head> area. This css affects all similar elements on the web page where the css is coded <style> selector{rule; [rule; ...} [selector{rule; [rule; ...}]</style> <style> p {color:red; width:50px} h1 {font-style:itialic}</style>
Link (styleSheet) this is the prefeered method of doing css. You place your css in a seperate physichal file. This file can be then used on ANY web page in your website. The link to this file will be done in your <head> area. The css in this file is similar to embedded EXCEPT there is no style tag selector{rule; [rule; ...} [selector{rule; [rule; ...}] p {color:red; width:50px} h1 {font-style:itialic}
